Tetris Classic 2023

(c) 2023 Renat Salimov


Controls:

up 	- rotate figure;
down 	- intent figure to down;
left 	- move figure to left;
right 	- move figure to right;
space	- drop figure down;

p	- pause;
a	- about this game.


Apart standart cursor keys, this game also works with cursor keys on numerical keypad.
